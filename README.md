# ARC_Template_Generator

## 開発
### Pythonでローカルサーバーを立てる

Python3をインストール後、`ARC_Template_Generator` ディレクトリ内で以下のコマンドを実行

```bash
$ python -m http.server
```

ブラウザで `localhost:8000` を開く
